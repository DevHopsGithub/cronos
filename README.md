Cronos is a small collection of functions to handle unix time and datetime in dart.

## Installing

1.  Add dependencies to `pubspec.yaml`

    ```yaml
    dependencies:
        cronos:
            git:
                url: https://github.com/DevHopsGithub/cronos
    ```

2.  Run pub get.

    ```shell
    flutter pub get
    ```

3.  Import package.

    ```dart
    import 'package:cronos/cronos.dart';
    ```
